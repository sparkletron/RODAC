uint8_t  vdpMode,
uint8_t  backColor 
)

Initialize TMS99XX struct with ports to use for input output, must match direction registers above. Register 1 has VRAM set to 16k, VDP mode set, the blank enabled(0), and all other bits in there disabled state. Memory addresses will match defines. These could be changed latter and then the setTMS99XXmode run for custom settings on the fly. Better option is to change the defines in this headed to your use case.

the below includes define other tms stuffs see them for more info METHODS

Parameters
p_tms99XXpointer to struct to contain data.
vdpModeset or change the mode, 0 = Graphics I, 1 = Graphics II, 2 = bitmap, 4 = Text.
backColorset background color to a 4 bit value.
